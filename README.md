# Expense Tracker API

## Setup
1. Copy `.env.example` to `.env` and fill in values.
2. `npm install`
4. `npm run dev` to start the server

## CORS
- CORS is enabled for origins listed in `CLIENT_URL` in `.env`.

## Scripts
- `npm start` - production server
- `npm run dev` - development with nodemon
- `npm test` - run tests
